# Translation Wrapper

Purpose
- To create a translation wrapper tool for a creative content team in telecommunciations company. They utilize this tool for translations to help generate content for the company's website, or new campaigns.

As-is Flow
- Copywriters come out with the content for a new campaign/ start a new language content.
- They need help in translating it into multilingual.
- They use AI for translaitons, but they dont sound natural or very intuitive.
- All AI translations needs human approval before being used.

To-Be Flow
- Copywriters utilize the wrapper to generate more contextual-suitable content.
- They will make translations with the help of prompting and AI-assistance.

# Supported Languages
- English (Main Source)
- Bahasa Malaysia
- Simplified Chinese

# Technology Stack
- Frontend: React + Vite + TailwindCSS + ShadCN
- Backend: Firebase (Firestore, Auth)
- AI: Google Gemini API (@google/genai)
- Files: xlsx library (Excel import/export)

# Core Features
## AI-ASSISTED TRANSLATION
- Gemini API integration
- Batch processing: 10 rows per API call
- Rate limit handling: exponential backoff on HTTP 429
- Queue management: stop/cancel functionality
- Dynamic model selection via VITE_GEMINI_MODEL env var
- Current model: gemini-2.0-flash

## Review Work
User Roles
- Guest: Can only view projects and glossary.
- Editor: Can create projects, do translations, edit translation, regenerate translations, but need approval in order to be used
- Admin: Manage User roles and settings configurations
- Manager: Has all the access of the above. Can approve/reject translations.

User Flow
- User enter Project Page through sidebar.
- This is a page that stores all the centralizes glossaries that are used throughout all projects.
- Users can choose to add new terms on their own or import form excel sheet.
- Similarly, users can utilize AI for multilingual translation and go through the review workflow again.
- Once approved, terms will be used throughout the translation in all projects to maintain consistency.

Approval Page
- Managers can utilize this page to bulk review and approve translations form both projects and glossaries.
